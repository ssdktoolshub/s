🚀 Quick Start

SSDK Tools Hub is a database-driven multi-tool platform. To add a new tool to the system, follow these three simple steps:

  1. Create your tool page HTML template inside /tools/tool-id.html.
  2. Write your tool business logic module inside /modules/tool-id.js extending the SSDKTool base class.
  3. Register your tool metadata inside /json/tools.json using the Admin Dashboard.

🛠 Base Tool SDK

Every tool page uses the SSDKTool base class to inherit lifecycle hooks, event handling, and layout binders.

import { SSDKTool } from "../core/sdk.js";

export default class MyCustomTool extends SSDKTool {
  async init(toolEngine) {
    await super.init(toolEngine);
    console.log("Custom tool initialized!");
  }

  async run(toolEngine) {
    this.showProgress(true, "Processing data...");
    
    // Grab value from toolInput field
    const input = this.getInputValue();
    
    // Business logic processes
    const processed = input.trim().split("").reverse().join("");
    
    // Set value into toolOutput
    this.setOutputValue(processed);
    
    this.showProgress(false);
    this.showStatus("✅ Reversal completed successfully!");
  }
}

🔌 Core Sub-Engines

The global system orchestrator window.SSDKCore registers several sub-engines that can be queried from any custom module script:

  • config: Load and parse static JSON files (config.loadJSON("faq.json")).
  • theme: Toggle themes and variables (theme.toggleTheme()).
  • search: Run fuzzy searches against indexes (search.search("query")).
  • notification: Display status toasts on-the-fly (notification.show("Done!", "success")).
  • analytics: Log telemetry counts (analytics.logEvent("tool", "run", "reverser")).
  • firebase: Run Firestore syncing and auth state queries.

🐍 Python FastAPI APIs

For resource-heavy tasks (PDF conversions, file compression, image transformations), query the FastAPI backend using the python sub-engine:

const pythonEngine = this.engine.core.getEngine("python");

// Running JSON payloads
const result = await pythonEngine.runBackendTask("/v1/ai/summarize", {
  text: "Raw string inputs",
  ratio: 0.3
});

// Running file uploads streams
const files = document.getElementById("file-upload").files;
const uploadRes = await pythonEngine.uploadFilesTask("/v1/pdf/merge", Array.from(files), (pct) => {
  console.log(`Upload progress: ${pct}%`);
});
