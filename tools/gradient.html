Using Gradients in CSS

Gradients are classified as images in CSS. Therefore, you should apply them to the background or background-image attributes rather than standard color rules:

.my-cool-container {
  background: linear-gradient(135deg, #ff9a9e 0%, #fecfef 100%);
}

Frequently Asked Questions

What is the difference between Linear and Radial gradients?＋
A Linear gradient shifts color along a straight line defined by an angle (e.g. 135 degrees), whereas a Radial gradient radiates outward from a central focal point in circular or elliptical patterns.
Do gradients affect page load performance?＋
No. Since CSS gradients are rendered natively by the browser's graphics engine, they are extremely efficient and load instantly without downloading heavy external graphic image assets.
