Advanced JSON Operations

  • Syntax-Highlighted Output: Keys, strings, numbers, booleans, and nulls rendered in distinct colours with no external dependencies.
  • JSON Diff Viewer: Deep recursive comparison between two JSON objects — green for added, red for removed, yellow for changed values.
  • JSONPath Query: Dot-notation + array-index path parser (e.g. $.developer.skills[0]) to extract nested values instantly.
  • JSON Schema Generator: One-click Draft-07 schema generation that recurses all types, properties, and array items.
  • Find & Replace: Recursively walk the JSON tree and replace every value for a given key, then see the highlighted result immediately.
  • Statistics Panel: Total keys, max nesting depth, type counts (arrays, strings, numbers, booleans, nulls) and byte size — shown after every parse.
  • Interactive Tree Viewer: Browse nested arrays and objects with collapsible caret nodes.
  • Export: Convert parsed JSON to XML or YAML and download as a file.

Frequently Asked Questions

How does the JSON Diff Viewer work?＋
Paste two JSON objects in the Diff tab and click Compare. A recursive deep-diff function walks both objects, annotating every key as added (in B but not A), removed (in A but not B), changed (value differs), or unchanged — then renders colour-coded output.
What JSONPath syntax is supported?＋
The built-in parser supports the $ root anchor, dot-separated property names, and square-bracket integer array indices — for example $.developer.skills[1] or $.features[0].name. Wildcards and filter expressions are not yet supported.
What does the Schema Generator produce?＋
It generates a valid JSON Schema (Draft-07) object with $schema, type, and properties. For arrays it adds an items sub-schema inferred from the first element. The schema can be downloaded as a .json file.
Is any data sent to a server?＋
No. Every operation — formatting, diff, schema generation, find & replace, path queries, syntax highlighting — runs entirely in your browser. Nothing leaves your device.
