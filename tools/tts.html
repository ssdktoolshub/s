⬇ Audio Download?
The Web Speech API synthesizes audio directly in the browser and does not produce a downloadable file. To save the audio, use a screen recorder (e.g. OBS Studio, Windows Xbox Game Bar Win+G, or macOS QuickTime) while the speech plays.

Key Features

  • Voice Library: Dynamically lists all built-in system speech voices with language filtering for English, Hindi, French, Spanish, German, and Japanese.
  • Word-by-Word Highlighting: A live highlight tracks each spoken word over the textarea using the browser's onboundary event.
  • Reading Progress Bar: A thin bar under the text fills as each word is spoken, giving you an at-a-glance read progress indicator.
  • Character & Word Stats: See instant word count, character count, and estimated reading time at the current speed — updated as you type.
  • Volume, Rate & Pitch Sliders: Fine-tune all three audio properties with real-time value readouts.
  • Sample Texts: One-click load for a News Headline, Poem Verse, or Tech Article Intro to test voices instantly.
  • 100% Serverless: Audio synthesis occurs entirely inside your browser — no data leaves your device.
