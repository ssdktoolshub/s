Frequently Asked Questions

What is a Base64 Data URI?＋
A Data URI is a Base64 string prefixed with metadata like `data:image/png;base64,`. This prefix informs the browser how to interpret the binary stream, allowing you to embed files directly inside HTML or CSS configs.
Is the file data safe?＋
Yes. Your files are processed entirely client-side. The file binary is read locally in your browser memory and converted without transferring any bytes over the network.
