📚 Clinical Pathology Notes: Gamma-Glutamyl Transferase

What is GGT?
Gamma-Glutamyl Transferase (GGT) is a microsomal membrane-bound enzyme found mostly in liver cells and bile ducts. Because it is highly sensitive to bile duct injury, it rises rapidly in cases of cholestasis.

How is it used alongside ALP?
ALP rises in both bone diseases and liver diseases. If ALP is high, checking GGT helps differentiate the source: if GGT is also high, the issue is hepatic (biliary); if GGT is normal, the issue is likely skeletal (bone).
