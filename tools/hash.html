Key Features

  • Text & File Hashing: Switch between typing text directly or uploading any file — binary content is read via FileReader and converted to a CryptoJS WordArray for accurate hashing.
  • HMAC Mode: Enter a secret key to compute HMAC-MD5, HMAC-SHA1, HMAC-SHA256, and HMAC-SHA512 keyed message authentication codes instead of plain hashes.
  • Hash Verification: Paste any known hash into the Verify field. The tool auto-detects the algorithm by length and shows a ✅ Match or ❌ Mismatch result instantly.
  • HEX / Base64 Toggle: Each hash row has its own format toggle — switch between hexadecimal (default) and Base64-encoded output per algorithm.
  • Copy as JSON: Export all four computed hashes as a single JSON object with one click.
  • Strength Indicators: Colored pills next to each algorithm label signal security strength: MD5 = Weak, SHA-1 = Legacy, SHA-256 = Strong, SHA-512 = Maximum.
  • 100% Client-side: All computation happens in your browser via CryptoJS. No data leaves your device.
