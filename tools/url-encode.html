Key Features

  • URL Encode/Decode: Converts strings using encodeURIComponent / decodeURIComponent (RFC 3986).
  • Base64 Encode/Decode: Fast browser-native Base64 using btoa / atob with UTF-8 support.
  • HTML Entity Encode/Decode: Escapes < > & " ' and unescapes them back safely.
  • Live Analysis: Real-time character count, special-char percentage, and output length delta as you type.
  • Auto-detect Mode: Click Encode and let the tool suggest the best mode for your input.
  • Swap: Instantly move output back to input for chained operations.
