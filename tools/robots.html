Frequently Asked Questions

What is a Robots.txt file?＋
A robots.txt file is a text file webmasters create to instruct web robots (typically search engine crawlers) how to crawl pages on their website. It is part of the Robots Exclusion Protocol (REP).
Does robots.txt hide pages from the general public?＋
No. Robots.txt is purely a set of instructions for search crawlers. It does not enforce password protection or restrict users from directly opening a URL in their web browsers. To secure pages, you must use authentication or server protection methods.
