Design Palette Types

  • Monochromatic: Adjusts lightness values of a single hue — great for clean, consistent UI designs.
  • Analogous: Adjacent colors on the 360° color wheel for harmonious, low-contrast blends.
  • Complementary & Triad: Colors opposite (180°) or at 120° intervals for vibrant, high-contrast contrasts.
  • CMYK: Cyan-Magenta-Yellow-Key (Black) model used in print design. Computed from RGB values.
  • WCAG Contrast: AA requires ≥4.5:1 (normal) or ≥3:1 (large text). AAA requires ≥7:1 (normal).
