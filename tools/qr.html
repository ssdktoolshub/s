Frequently Asked Questions

Will my QR code scan if I add a logo?＋
Yes. We default error correction to "H (30%)" which repeats encoded bits multiple times. This allows mobile cameras to successfully reconstruct and scan the code even with a logo covering the center patterns.
What does the dot style do?＋
After qrcodejs renders the QR matrix to a canvas, we read every dark pixel and redraw each module (cell) in your chosen shape — square, rounded rectangle, or circle. All standard scanners can decode any of these styles.
How does Test Scan work?＋
It reads the generated canvas pixel data and passes it to the jsQR library which runs a full QR decode in the browser — showing you exactly what a camera would decode, without any network request.
Can I change colors to anything?＋
Yes, but maintain high contrast. Dark dots on a light background work best. If the colors are too similar, scanners will fail to capture the grid.
