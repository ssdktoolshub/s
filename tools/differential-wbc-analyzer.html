📚 Clinical Pathology Notes: WBC Differential

What is a WBC Differential?
A WBC differential measures the percentage of each type of white blood cell in your blood. There are five main types: Neutrophils, Lymphocytes, Monocytes, Eosinophils, and Basophils. Each plays a distinct role in your immune defense.

What are the roles of each leukocyte type?
1. **Neutrophils (40-70%)**: Target bacterial infections first.
2. **Lymphocytes (20-40%)**: Target viral infections (produces antibodies, includes T-cells/B-cells).
3. **Monocytes (2-8%)**: Clean up dead tissues and target chronic infections.
4. **Eosinophils (1-4%)**: Defend against parasites and participate in allergic reactions.
5. **Basophils (0.5-1%)**: Release histamine during severe allergic responses.

How is absolute count determined?
Absolute count measures the actual number of cells per microliter. It is computed as: Absolute Count = (WBC Count × Differential %) / 100. This is vital for checking severe immunodeficiencies (like neutropenia).
