Frequently Asked Questions

Does Python run on a backend server?＋
No. Standard setups require hosting a backend service. This tool loads Python directly into WebAssembly (WASM), processing code inside your browser thread. This guarantees absolute data privacy and keeps server load at zero.
Can I import third-party packages?＋
Yes. By choosing a package in the dropdown, the browser automatically fetches and installs the compiled WebAssembly package wrapper, allowing standard imports (e.g. import numpy as np) to execute correctly.
