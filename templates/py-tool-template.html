Developer Quick Start Guide

To run Python scripts directly inside HTML/JS templates without any server overhead, follow these instructions:

  • Load Pyodide: Include the Pyodide WebAssembly script tag inside the document <head>.
  • Initialize Runtime: Invoke loadPyodide() in Javascript. This fetches the core Python interpreter WASM binaries asynchronously.
  • Bridge Variables: Use pyodide.globals.set("variable_name", js_value) to pass elements into Python.
  • Fetch Outputs: Access variables inside Python using pyodide.globals.get("variable_name") in JS after running scripts.
